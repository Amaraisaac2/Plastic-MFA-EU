# Plastic-MFA-EU

Reconstructing plastic material flows across the EU economy using Eurostat PRODCOM data and JRC transfer coefficients.

## Project Structure

- `data/` : methodological input tables
- `notebooks/` : Jupyter workflows
- `outputs/` : generated MFA outputs and Sankey diagrams
- `figures/` : visualizations
- `src/` : reusable Python scripts

## Data Sources

- Eurostat PRODCOM
- JRC transfer coefficient methodology

## Methods

- Data cleaning
- Sector allocation
- Transfer coefficients
- Material flow balancing
- Sankey visualization
